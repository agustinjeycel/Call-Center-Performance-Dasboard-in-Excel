# CALL CENTER PERFORMANCE DASHBOARD	
	
## Project Overview	
	
This Excel dashboard analyzes call center performance using key operational metrics such as incoming calls, answered calls, answer rate, abandoned calls, waiting time, and service level.	
The dashboard helps monitor customer service performance and identify trends that may impact customer experience.	
	
	
	
## Objective	
	
The purpose of this project is to:	
	
Monitor call center activity	
Track answering performance	
Measure service efficiency	
Analyze customer waiting times	
Provide insights through an interactive Excel dashboard	
	
	
	
## Dataset Information	
	
The dataset contains call center performance records with the following fields:	
	
Column	Description
Incoming Calls	Total calls received
Answered Calls	Total calls answered
Answer Rate	Percentage of calls answered
Abandoned Calls	Calls abandoned before being answered
Answer Speed (AVG)	Average time to answer a call
Talk Duration (AVG)	Average call duration
Waiting Time (AVG)	Average customer waiting time
Service Level (20 Seconds)	Percentage of calls answered within 20 seconds
	
	
## Data Preparation	
	
The following steps were performed before creating the dashboard:	
	
Checked for missing values	
Verified data types	
Formatted percentage fields	
Formatted time-related columns	
Converted the dataset into an Excel Table	
	
	
## Key Performance Indicators (KPIs)	
	
Total Incoming Calls	
Total number of calls received.	
	
Total Answered Calls	
Total number of calls successfully answered.	
	
Average Answer Rate	
Percentage of incoming calls answered.	
	
Total Abandoned Calls	
Number of calls abandoned before being answered.	
	
Average Waiting Time	
Average time customers waited before connecting to an agent.	
	
Average Service Level	
Percentage of calls answered within 20 seconds.
